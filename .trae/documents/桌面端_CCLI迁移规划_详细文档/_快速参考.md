# 快速参考

## 常用命令

### 开发命令

```bash
# 桌面端开发
npm run tauri dev

# CLI 开发
cd src-tauri
cargo run --bin ai-ide-cli -- --help

# 完整构建
npm run tauri build
```

### CLI 子命令

```bash
# 帮助
ai-ide-cli --help

# 初始化工作区
ai-ide-cli init ./my-project

# AI 对话
ai-ide-cli chat "帮我写一个 Hello World"
ai-ide-cli chat "帮我优化这段代码" --no-stream

# 编辑文件
ai-ide-cli edit ./src/main.rs
ai-ide-cli edit ./src/main.rs --line 10 --column 5

# 运行命令
ai-ide-cli run godot -- --version
ai-ide-cli run npm -- run dev --cwd ./my-project

# 检查环境
ai-ide-cli check

# 交互模式
ai-ide-cli repl
```

## Tauri IPC 调用示例

```typescript
import { invoke } from '@tauri-apps/api/tauri';

// 调用 Rust 命令
const content = await invoke<string>('read_workspace_file', {
  workspacePath: '/path/to/workspace',
  relativePath: '/src/main.ts'
});

// 列出文件
const files = await invoke<FileEntry[]>('list_workspace_files', {
  workspacePath: '/path/to/workspace',
  prefix: '/src'
});

// 搜索
const hits = await invoke<SearchHit[]>('search_in_workspace', {
  workspacePath: '/path/to/workspace',
  query: 'function',
  maxMatches: 50
});
```

## Rust Command 注册

```rust
// src-tauri/src/lib.rs

#[tauri::command]
fn read_workspace_file(workspace_path: String, relative_path: String) -> Result<String, String> {
    // 实现
}

fn main() {
    tauri::Builder::default()
        .invoke_handler(tauri::generate_handler![
            read_workspace_file,
            write_workspace_file,
            list_workspace_files,
            search_in_workspace,
            run_command,
            // ... 其他命令
        ])
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

## 环境变量

| 变量 | 说明 | 必需 |
|------|------|------|
| `DEEPSEEK_API_KEY` | DeepSeek API 密钥 | 是 |
| `RUST_LOG` | Rust 日志级别 | 否 |
| `TAURI_DEBUG` | 启用调试模式 | 否 |
